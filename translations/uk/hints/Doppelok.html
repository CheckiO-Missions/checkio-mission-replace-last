Перш за все, прочитай про методи списків "list.pop()" та "list.insert()" за посиланням .
Я все ще не розумію... Потрібна допомога, що з цими методами робити?
Метод "list.pop()" прибирає елемент на конкретній позиції із списку, позицію потрібно вказувати у дужках "()". Якщо ти впишеш в консолі "a = [1, 2, 3]", а потім використаєш метод "a.pop(1)", ти отримаєш "[1, 3]". 
a = [1,2,3]
a.pop(1)
print(a) # output: [1, 3]
Так, мені зрозуміло, як це працює, а далі що?
Використовуй метод "list.insert()" щоб додати елемент у список на конкретну позицію. Таким чином, якщо ти впишеш у консоль "list.insert(0, 2)", то отримаєш "[2, 1, 3]". 
a = [1,3]
a.insert(0, 2)
print(a) # output: [2,1,3]
Останнє запитання. Як має виглядати мій фінальний код?

Твій код повинен мати наступний вигляд. А також, не забудь про перевірку на те, чи порожній список.

def replace_last(items: list) -> list:
    if items:
        items.insert(0, items.pop(-1))
    return items
